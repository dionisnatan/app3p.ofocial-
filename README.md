# Instagran App3P — Landing Page Oficial

Landing page profissional da **Instagran App3P**, desenvolvida para converter visitantes em clientes.

## 🚀 Tecnologias

- HTML5 semântico
- CSS3 com glassmorphism, animações aurora e partículas
- JavaScript vanilla (sem dependências)
- Fontes Google: Playfair Display + Space Grotesk

## ✨ Recursos

- Design dark premium com paleta dourada
- Animações aurora e partículas flutuantes
- Cards com efeito glassmorphism
- Navbar com scroll suave e backdrop blur
- Seções: Hero, Stats animados, Benefícios, Projetos, Como Funciona, Preço, Depoimentos, CTA
- Totalmente responsivo (mobile-first)
- Integração direta com WhatsApp

## 📁 Arquivos

| Arquivo | Descrição |
|---|---|
| `pagina.html` | Página principal completa |
| `vercel.json` | Configuração de deploy na Vercel |

## 🌐 Deploy

Hospedado na **Vercel** via integração com GitHub.
